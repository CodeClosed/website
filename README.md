# 👨‍💻 Personal Portfolio Website

![Project Status](https://img.shields.io/badge/Status-Active-success)
![License](https://img.shields.io/badge/License-MIT-blue)

A modern, responsive personal portfolio website designed to showcase my projects, technical skills, and academic journey as a Computer Science student (AI & ML). The site features a **Glassmorphism** UI style, dark/light mode toggling, and smooth scroll animations.

---

## 🚀 Live Demo

[**View Live Demo**](http://www.vihaanvarshney.com/#home)

---

## ✨ Features

* **🎨 Glassmorphism Design:** Modern UI with frosted glass effects, blurred backgrounds, and vibrant gradients.
* **🌓 Dark/Light Mode:** Fully functional theme toggle with local storage persistence (remembers your preference).
* **📱 Fully Responsive:** Optimized for desktops, tablets, and mobile devices with a custom hamburger menu.
* **⚡ Smooth Animations:** Elements fade in and slide up as you scroll down the page.
* **🎓 Education Timeline:** A vertical timeline showcasing academic milestones with years and GPAs.
* **🛠 Skill Pills:** A clean grid layout displaying technical proficiencies.

---

## 🛠️ Tech Stack

* **Frontend:** HTML5, CSS3 (Custom Variables, Flexbox, Grid), JavaScript (ES6).
* **Design:** CSS Glassmorphism effects (`backdrop-filter`), Gradient text.
* **Fonts:** [Outfit](https://fonts.google.com/specimen/Outfit) (Google Fonts).

## 🚀 Getting Started

To view or modify this project locally, follow these steps:

### 1. Clone the repository

```bash
git clone [https://github.com/CodeClosed/portfolio.git](https://github.com/CodeClosed/portfolio.git)

```

### 2. Navigate to the folder

```bash
cd portfolio

```

### 3. Run the project

Simply open **`index.html`** in your web browser.

*Recommended:* If you use VS Code, install the **Live Server** extension. Right-click `index.html` and select "Open with Live Server" for real-time updates.

---

## 🎨 Customization

### Changing Colors

Open `style.css` and modify the CSS variables in the `:root` selector to change the color scheme globally.

```css
:root {
  --accent: #2563eb;       /* Main accent color */
  --gradient: ...;         /* Change the gradient colors here */
}

```

### Changing Images

Replace `hero-bg.png` with any high-quality image of your choice. Ensure you update the filename in `style.css` if the name changes.
